# Python Poetry (Dependency & Package Manager)

If you are building ML pipelines, Poetry is a tool that replaces outdated requirements files (pip, venv, conda). It automatically handles complex ML dependency conflicts (e.g., matching PyTorch, TensorFlow, and CUDA versions) and enforces project reproducibility through a strict pyproject.toml file.

---

### How it helps in ML:

- **Reproducibility:** Prevents the dreaded "it works on my machine" issue by locking specific package versions.
- **Clean Environments:** Automatically creates and manages isolated virtual environments for your ML experiments.
- **Easy Publishing:** Allows you to easily package and publish custom model utilities or deployment code.
